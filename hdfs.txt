when ever we fire hdfs/hadoop command it will iteract with namenode
    hadoop fs / hdfs dfs (internally it calls hadoop fs only) - list of all commands available
    hadoop fs -ls - home directry
    hadoop fs -mkdir /user/
    local to hdfs -
        1. hadoop fs -put <local file path> <hadoop path>
        2. copyFromlocal
        3. hadoop fs -rm
        4. hadoop fs -cat to check the data
    hdfs to local - 
        1. hadoop fs -get or copyToLocal
        5. hadoop fs -mkdir
        6. hadoop fs -mv //move
        7. hadoop fs -cp //copy with in hdfs
        8. hadoop fs -ls -h (human readable file size)
